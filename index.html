Questions:

Four questions from #24 - #38:

    Match the appropriate software concepts to their respective generations.

    1. Assemblers

    2. Operating systems

    3. Word processors

    4. C++

Answer:

    1. Assemblers are part of the first generation of software. Assemblers also mark the beginning of true abstraction in computing.

    2. Operating systems were introduced in the third generation of software. Operating systems expanded the gap between the user and hardware.

    3. Word processing was seen with the fourth generation of software. Furthering abstraction.

    4. C++ was introduced just after the popular programming language C in the fourth generation of software. Largely regarded as one of the best programming languages still used today C++ is a structured language that allows low-level control.
